# FoodHub Order Analysis
Exploratory Data Analysis of FoodHub food delivery orders using Python

## Project Overview
Exploratory Data Analysis of 1,898 food delivery orders 
from the FoodHub platform using Python.

## Objective
Analyze order patterns to identify key insights and provide 
actionable business recommendations.

## Key Findings
- American and Japanese cuisines are most popular
- Shake Shack is the most ordered restaurant
- Weekends drive 2.5x more orders than weekdays
- 10.54% of orders exceed 60 minutes total delivery time
- Net revenue generated: $6,166.30

## Technologies Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Google Colab
